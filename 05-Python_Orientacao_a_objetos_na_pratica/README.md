# Python - Orientação a objetos na prática

https://www.schoolofnet.com/curso/python/linguagem-python/python-orientacao-objetos-na-pratica/

Descrição: Neste curso, vamos aprender na prática a usar o Orientação a Objetos da linguagem Python na solução de um problema. Aprenderemos boas práticas da Orientação a Objetos, novos conceitos de OO e a como pensar na implementação do código-fonte a medida que a aplicação vai se modificando e crescendo.
